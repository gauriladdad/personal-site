A premium, minimalist news site designed specifically for kids. It features a hybrid AI pipeline that combines reliable reporting from world-class sources with smart, kid-friendly AI summaries.

Live at: [news.gattani.ca](https://news.gattani.ca)

## Core Features

- **Hybrid AI News Pipeline**:
  - **The Reporter**: Fetches reliable, up-to-the-minute news from BBC and CBC RSS feeds.
  - **The Editor**: Uses **Google Gemini 1.5 Flash** to rewrite complex stories into simple, 2-3 sentence summaries suitable for an 11-year-old.
  - **AI Generation**: Custom categories (like **Canada**) are fully reported and generated by Gemini to provide localized, high-interest stories.
- **AI-Driven Vetting**: All stories are automatically evaluated by Gemini for safety, child-friendliness, and relevance.
- **Smart Insights**: Every story includes an AI-generated "Why it matters" bullet point to help kids understand the importance of the news.
- **Modern UI/UX**:
  - **Theme Switcher**: Beautifully integrated Light and Dark modes with persistent storage.
  - **Centered Layout**: Professional, fixed-width layout (`900px`) for maximum readability on all devices.
  - **Responsive Design**: Smooth horizontal navigation that works perfectly on mobile and desktop.
- **Optimized for Growth**:
  - **SEO & Social**: Full Open Graph, Twitter Card, and Meta tag support for beautiful sharing.
  - **Google Analytics**: Integrated tracking (via `gtag.js`) to understand user engagement.

## Technology Stack

- **Frontend**: [Astro](https://astro.build/) (v4+) for zero-JS performance and static optimization.
- **Backend**: AWS Lambda (Python 3.12) for automated daily news fetching and processing.
- **Storage**: AWS S3 for hosting JSON news data.
- **AI Engine**: Google Gemini 1.5 Flash via Google AI Studio.

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Backend Setup**:
   - The Lambda source is in `/backend/lambda_function.py`.
   - Run `build_lambda.sh` to package dependencies and prepare for deployment.
   - Configure environment variables in AWS: `S3_BUCKET_NAME` and `GEMINI_API_KEY`.

## Deployment

- **Frontend**: Deployed on **Cloudflare Pages**. Build command: `npm run build`, Output directory: `dist`.
- **Backend**: Deployed on **AWS Lambda** with an EventBridge trigger for daily updates.
- **Content**: Served via **AWS S3** (public bucket or CloudFront).
